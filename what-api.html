What is API?

An API, or Application Program Interface, is a set of routines, protocols, and tools for building software applications. An API specifies how software components should interact. Additionally, APIs are used when programming graphical user interface (GUI) components.

There are many useful APIs out there that are available to the public at no cost. One example is the OpenWeatherMap API. It can help us get weather data based on a given city or zipcode so that one don't need to gather weather data themselves in order to embed the info on their application or website. In this guide, we are going to discuss the NASA APIs specifically. First of all, you can access the page by simply clicking the large image above or here.

The NASA API page provides us with many useful APIs that can be used to not only get spectacular images that are stored in their massive database, but also some other scientific data gathered over the past decades.

For the purpose of this guide, we will focus on just two most popular and exciting APIs: Astronomy Picture of the Day (APOD), and Earth Polychromatic Imaging Camera (EPIC). We will learn how to interact with them using JSON and Javascript so that they can provide us with images and details of the images based on our given search criteria.

What are API keys?

and why do we need them?

What are API keys?

An application programming interface key (API key) is a code passed in by computer programs calling an application programming interface (API) to identify the calling program, its developer, or its user to the Web site. You can get more detailed info about API keys here.

But why API keys?

Great question! Since most APIs are available for free, everyone can access and make requests to them. Just like the Earth, the servers hosting these APIs have limited resources and thus if someone with detrimental intention tries to sabotage the server, they can attempt to flood the servers with numerous repetitive requests that can backlog the servers and potentially render them inaccessible to other users. And there would be no way to track them down without the use of API keys.

Each API key will help uniquely identify each developer and keep track of the number of requests that are made to the servers in a specified timeframe. Once a developer has exceeded their limits of calls, the server will temporarily block them from making further calls. Then based on the frequency of those blocks, the API provider may decide to permanently block the user altogether if it is determined that the user has had malicious intents using the service. As API keys are important in identifying the user, you should treat them as your password to your online bank accounts and never share them. Unless you want to get banned out of a blue day for no reasons :-)

How to get an API key?

Since we are going to discuss the NASA APIs here, we will discuss how to get a NASA API key. But do we need a key to use their APIs? The answer is no. According to the site, you don't need to have a key to explore NASA data. If you are just trying out their APIs, you can use their generic key: DEMO_KEY. However, if you are going to use the service intensively, you should sign up for a NASA developer key. Once you have completed the form, an API key will be emailed to you within a few minutes. And remember to keep it secured!
